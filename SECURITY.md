# Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do not** open a public issue
2. Email the maintainers directly or reach out via the [primary noaabathymetry repository](https://github.com/noaa-ocs-hydrography/BlueTopo)
3. Include a description and steps to reproduce

We will acknowledge and assess within 7 days.
