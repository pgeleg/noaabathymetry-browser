# noaabathymetry-gui

Desktop GUI wrapper for the [noaabathymetry](https://github.com/noaa-ocs-hydrography/BlueTopo) Python package.

## Getting Started

1. Go to [Releases](../../releases)
2. Download the zip for your platform (macOS or Windows)
3. Unzip and run the application

No installation or Python environment required.
